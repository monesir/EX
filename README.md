# مؤرشف موقع الديوان الذكي (Aldiwan.net Scraper)

هذا البرنامج عبارة عن أداة متكاملة لتحميل وأرشفة جميع القصائد من موقع [الديوان](https://www.aldiwan.net/) بضغطة زر واحدة.

## المميزات
- **واجهة رسومية (GUI):** سهلة الاستخدام لاختيار مجلد الحفظ ومتابعة التقدم.
- **تخطي الحماية:** يستخدم مكتبة `cloudscraper` لتخطي حماية Cloudflare.
- **تنظيم ذكي للمجلدات:** يتم حفظ القصائد وتصنيفها كالتالي: `العصر أو الدولة / اسم الشاعر / القصيدة.txt`.
- **استخراج السيرة الذاتية:** يقوم السكربت بسحب نبذة عن الشاعر وحفظها في ملف `نبذة عن الشاعر.txt`.
- **تنسيق الشعر العمودي:** يتعرف على القصائد العمودية ويقوم بتنسيقها بشطرين (صدر * عجز).
- **حفظ التقدم:** عند إيقاف البرنامج وتشغيله مجدداً، سيتخطى الملفات المحملة مسبقاً ليكمل من حيث توقف.

## المتطلبات
- Python 3.7+
- تثبيت المكتبات المطلوبة عبر الأمر:
  ```bash
  pip install -r requirements.txt
  ```

## طريقة الاستخدام
1. قم بتشغيل الواجهة الرسومية:
   ```bash
   python app_desktop.py
   ```
2. اختر مسار الحفظ (مثلاً `D:\aldiwan\Poets`).
3. اضغط على **"بدء الأرشفة"** واترك البرنامج يعمل.

## الملفات
- `app_desktop.py`: النسخة الأساسية ذات الواجهة الرسومية.
- `aldiwan_crawler.py`: نسخة تعمل من سطر الأوامر (CLI) لمن يفضلها.
